## 2020

### An NFT That Saves Lives (nft.html)
- [ ] 未读
推广 Noora Health 的 NFT 拍卖来拯救南亚医院中的生命——每花费约1235美元就能拯救一条生命。
🔗 https://paulgraham.com/nft.html

### The Real Reason to End the Death Penalty (real.html)
- [ ] 未读
废除死刑因为至少4%的死囚是无辜的，错误判决来自警察/检察官不当行为、无能辩护律师、伪证和不可靠法医证据。
🔗 https://paulgraham.com/real.html

### How People Get Rich Now (richnow.html)
- [ ] 未读
财富来源从1982年的继承和自然资源到2020年的科技公司创业，代表回归历史常态，由放松管制、技术进步和创业成本下降推动。
🔗 https://paulgraham.com/richnow.html

### Write Simply (simply.html)
- [ ] 未读
简单写作让想法自由流入读者心智，减少摩擦，展示对观众考虑，且经得起时间考验。
🔗 https://paulgraham.com/simply.html

### Donate Unrestricted (donate.html)
- [ ] 未读
限制性捐赠天生是次优的，因为假设捐赠者比非营利组织更知道如何分配资金。"如果你信任他们花你的钱，就信任他们决定如何花。"
🔗 https://paulgraham.com/donate.html

## 2021

### A Project of One's Own (own.html)
- [ ] 未读
为自己做项目比被分配工作更有成效，成年人应保持这种方法。
🔗 https://paulgraham.com/own.html

### Fierce Nerds (fn.html)
- [ ] 未读
"激烈的书呆子"是有高度效率的问题解决者，但需要野心勃勃的工作来避免痛苦。
🔗 https://paulgraham.com/fn.html

### Crazy New Ideas (newideas.html)
- [ ] 未读
我们应该谨慎对待 dismissing 来自合理领域专家的"疯狂"想法，因为这些提议极可能有价值，最明智的回应是提问而不是 dismiss。
🔗 https://paulgraham.com/newideas.html

## 2022

### Founder Mode (foundermode.html)
- [ ] 未读
创始人被错误建议像职业经理人一样运营公司（"雇用优秀人才并给他们空间"），但应该采用"创始人模式"更直接地参与整个组织。
🔗 https://paulgraham.com/foundermode.html

### The Right Kind of Stubborn (persistence.html)
- [ ] 未读
区分"坚持"（正确的固执）和"固执"（错误的固执）：坚持者保持目标附着同时调整方法，固执者无论证据如何都坚持最初想法。
🔗 https://paulgraham.com/persistence.html

### When To Do What You Love (when.html)
- [ ] 未读
是否追随激情取决于财富目标：如果你想成为超级富豪（且年轻有技术），追随真正感兴趣的事变得有利，因为它自然导向创业想法。
🔗 https://paulgraham.com/when.html

### What I've Learned from Users (users.html)
- [ ] 未读
Y Combinator 资助数千家创业公司的经验：大多数创业公司共享常见问题；创始人常因创业的反直觉性而错误识别真正挑战；成功取决于专注、速度和向有经验的顾问及同行创始人学习。
🔗 https://paulgraham.com/users.html

### Heresy (heresy.html)
- [ ] 未读
异端概念以世俗形式回归——被当作犯罪的观点，其权重超过真相和演讲者的其他成就。
🔗 https://paulgraham.com/heresy.html

### Putting Ideas into Words (words.html)
- [ ] 未读
把想法写成文字是"严峻考验"，揭示思维中的空白，迫使我们形成更完整的理解。"没有写过某个话题的人对该话题没有完全形成的想法。"
🔗 https://paulgraham.com/words.html

### Is There Such a Thing as Good Taste? (goodtaste.html)
- [ ] 未读
好品味确实存在，通过归谬法证明——如果没有好品味，就不可能有好艺术或熟练艺术家。
🔗 https://paulgraham.com/goodtaste.html

### Beyond Smart (smart.html)
- [ ] 未读
智慧与产生重要新想法不同——许多聪明人成就很少。智慧与创新之间的差距包含可培养的成分，如强迫性兴趣、独立思考、写作能力和良好的工作习惯。
🔗 https://paulgraham.com/beyondsmart.html

### Weird Languages (weird.html)
- [ ] 未读
大多数编程（99.5%）只是"粘合库调用"，但0.5%是最有趣技术所在。学习奇怪语言帮助程序员发现新的思维方式。
🔗 https://paulgraham.com/weird.html

### How to Work Hard (hwh.html)
- [ ] 未读
要做出伟大成就，需要天赋与努力的结合。关键是找到适合自己的领域，在不过度消耗的前提下尽可能多地投入时间。
🔗 https://paulgraham.com/hwh.html

## 2024

### Writes and Write-Nots (writes.html)
- [ ] 未读
AI将把写作者分为"会写的"和"不会写的"两极，消除中间地带。写作即思考，这可能造成一个思想清晰与模糊分化的世界。
🔗 https://paulgraham.com/writes.html

### The Brand Age (brandage.html)
- [ ] 未读
瑞士手表通过成为奢侈品牌在石英危机中幸存，说明技术商品化推动行业进入"品牌时代"。
🔗 https://paulgraham.com/brandage.html

### The Shape of the Essay Field (field.html)
- [ ] 未读
探讨随笔领域的现状和演变。
🔗 https://paulgraham.com/field.html

### Good Writing (goodwriting.html)
- [ ] 未读
好文章听起来好，更可能是对的——润色句子能下意识修正想法，流畅的文章让错误更容易被发现。
🔗 https://paulgraham.com/goodwriting.html

### What to Do (do.html)
- [ ] 未读
人生三原则：帮助他人、照顾世界、创造好的新事物。其中"创造好的新事物"是人类充分发挥潜能的方式，也是最稀缺的贡献。
🔗 https://paulgraham.com/do.html

### The Reddits (reddits.html)
- [ ] 未读
回顾 Reddit 创始人被 YC 拒绝后又获得资助，最终创建 Reddit 的故事。
🔗 https://paulgraham.com/reddits.html

### How to Start Google (google.html)
- [ ] 未读
要想创办成功的创业公司，年轻人需要通过项目精通技术，同时在学校取得好成绩进入顶尖大学找到志同道合的联合创始人。
🔗 https://paulgraham.com/google.html

### The Best Essay (best.html)
- [ ] 未读
最好的随笔需要重要话题+惊人发现，通过发现式写作遵循问题路径+广泛性和严谨修订来实现。
🔗 https://paulgraham.com/best.html

### Superlinear Returns (superlinear.html)
- [ ] 未读
回报是超线性的而非线性的，源于指数增长和赢家通吃效应，理解这个概念对追求成功至关重要。
🔗 https://paulgraham.com/superlinear.html

### How to Do Great Work (greatwork.html)
- [ ] 未读
做伟大工作的四个步骤：找到与天赋兴趣匹配的工作、学习达到前沿、注意知识空白、探索有希望的领域。好奇心是做伟大工作的引擎和舵。
🔗 https://paulgraham.com/greatwork.html

### How to Get New Ideas (getideas.html)
- [ ] 未读
产生新想法的关键是注意异常——看起来奇怪、缺失或破损的东西。最肥沃的发现领域是知识前沿。
🔗 https://paulgraham.com/getideas.html

### The Need to Read (read.html)
- [ ] 未读
阅读不仅是获取信息，更是提升写作和思维能力的关键。写作不仅是表达思想的方式，更是产生新思想的过程。
🔗 https://paulgraham.com/read.html

### What You (Want to)* Want (want.html)
- [ ] 未读
探讨自由意志与欲望的关系：我们能控制基于欲望的行动，但总有一些层次的"想要"（如"想要想要"）是我们最终无法控制的。
🔗 https://paulgraham.com/want.html

### Alien Truth (alien.html)
- [ ] 未读
提出"外星真理"概念——超越数学和物理的普遍原则，所有智能生命都会认同，这应该是哲学的真正目标。
🔗 https://paulgraham.com/alien.html
