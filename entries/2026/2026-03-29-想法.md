#AI协作 #飞书

今天在 AI 工作流里集成了 larksuite/cli，把飞书日历、代办、消息都接进来了。

安装配置过程：
1. npm install -g @larksuite/cli
2. npx skills add larksuite/cli --all -y -g
3. lark-cli config init --new 配置应用
4. lark-cli auth login --recommend 授权

验证了发消息、创建日历、创建代办都能用。感觉以后日程和任务管理可以直接用飞书了，不用再另外维护一套。
