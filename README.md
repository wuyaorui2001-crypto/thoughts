## 核心理念

**你只管输入，AI 负责归档。**

不用想文件名、不用选分类、不用打标签。直接说/写你的想法，剩下的交给我。

---

## 使用方式

### 方式1：直接输入（推荐）

```
你：今天看了《星际穿越》，那个五维空间的呈现方式...

我：已记录 → 自动打标签 [#影片 #观后感 #科幻] → 存入 2026-03-17.md
```

### 方式2：明确指定

```
你：记录灵感：做一个能自动整理笔记的AI工具...
你：记日记：今天心情很低落...
```

## 项目结构

```
thoughts/
├── entries/              # 所有原始条目（按天聚合）
│   ├── 2026-03-17.md    # 今天的所有想法
│   ├── 2026-03-16.md    # 昨天的
│   └── ...
├── index/by-tag/         # 标签索引（AI自动维护）
│   ├── 灵感.md          # 所有灵感条目
│   ├── 影片.md          # 所有观影记录
│   ├── 日记.md          # 所有日记
│   └── ...
└── reviews/              # 定期回顾（AI生成）
    ├── weekly/          # 周回顾
    └── monthly/         # 月回顾
```

## 标签体系（AI自动识别）

| 标签 | 说明 | 示例 |
|------|------|------|
| #灵感 | 创意、想法、脑洞 | "做一个AI记账软件..." |
| #日记 | 心情、日常、感悟 | "今天心情不错..." |
| #影片 | 电影、剧集、纪录片 | "看完《XXX》感觉..." |
| #书籍 | 读书笔记 | "读《原则》第3章..." |
| #工作 | 项目、职业相关 | "今天遇到一个需求..." |
| #学习 | 知识、技能 | "学Python时发现..." |
| #健康 | 身体、心理 | "最近睡眠质量..." |
| #人际 | 关系、社交 | "和朋友聊天想到..." |
| #待验证 | 待测试的想法 | #灵感 #待验证 |
| #重要 | 高优先级 | #工作 #重要 |

## 特点

- ✅ **零负担记录** - 不用管格式、分类、文件名
- ✅ **自动标签** - AI根据内容智能打标签
- ✅ **时间线清晰** - 按天聚合，追溯方便
- ✅ **多维索引** - 按标签、时间、关联度检索
- ✅ **定期回顾** - 周/月总结自动生成
- ✅ **可迁移** - 纯Markdown，任何工具都能打开
