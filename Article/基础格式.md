| **项目**                 | **推荐数值**       | **理由**                                 |
| **字号 (Font Size)**     | **15px**       | 16px太笨重，14px太细碎，15px最显精致，适合长文。         |
| **行间距 (Line Space)**   | 1.75           | 增加上下行间距能显著降低阅读疲劳感。                     |
| **字间距 (Letter Space)** | 2 px           | **极其重要！** 默认的0px太挤，拉开1.5px会让文字呼吸感瞬间拉满。 |
| **两端缩进 (Margins)**     | **16 或 20**    | 两边留白，能像实体书一样把视线聚焦在屏幕中央。                |
| **段前/段后距**             | **15px**       | 建议段落之间不只是敲回车，设置段后距会让间距更均匀。             |
| **文字颜色**               | **#333333**    | 接近纯黑但更柔和，适合长时间盯着屏幕。                    |
| **背景颜色**               | **#FFFFFF**    | 纯白背景是永恒的经典，最衬文字。                       |
| 标题                     | EP 00x \| xxxx | 尽可能在4-5个字                              |
